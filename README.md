# Taller DevOps: Flask, PostgreSQL, pgAdmin y GitHub Actions

Este proyecto implementa una arquitectura multi-contenedor utilizando **Docker Compose** para orquestar una aplicación web en **Flask**, una base de datos **PostgreSQL**, y el administrador **pgAdmin**. Todo el proceso de integración y entrega continua (CI/CD) está automatizado con **GitHub Actions**.

---

## 🛠️ Requisitos de Ejecución (Para Entorno Local)

Para levantar este proyecto de forma local (en una máquina con Docker Desktop activo), se deben seguir los siguientes pasos:

### 1. Clonar el repositorio

```bash
git clone [https://github.com/20Anthony/devops-flask-postgres.git](https://github.com/20Anthony/devops-flask-postgres.git)
cd devops-flask-postgres
```
